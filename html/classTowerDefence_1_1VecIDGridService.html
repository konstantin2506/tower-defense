Detailed Description

Реализация IGridService, позволяющая хранить несколько ID в одной ячейке.

Использует матрицу векторов для хранения нескольких идентификаторов в каждой ячейке. Поддерживает многопоточный доступ при включенном флаге PARALLEL.
