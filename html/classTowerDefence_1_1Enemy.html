Detailed Description

Класс, представляющий врага в игре.

Является конкретной реализацией интерфейса IEnemy. Содержит компоненты для управления здоровьем, движением, эффектами и способностями. Использует паттерн "Стратегия" для атаки. Объект является неизменяемым (final).
