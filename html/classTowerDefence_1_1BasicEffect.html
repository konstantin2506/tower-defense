Detailed Description

template<typename OwnerT>
class TowerDefence::BasicEffect< OwnerT >

Базовый класс для всех временных эффектов.

Шаблонный класс, предоставляющий общую функциональность для эффектов: таймер длительности, множитель эффекта и управление состоянием завершения.
