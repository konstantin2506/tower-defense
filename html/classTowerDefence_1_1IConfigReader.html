Detailed Description

Интерфейс для чтения и записи конфигурационных данных.

Предоставляет абстракцию для работы с конфигурационными данными различных форматов (JSON, XML, YAML и т.д.). Поддерживает чтение примитивных типов, вложенных структур и запись значений.
