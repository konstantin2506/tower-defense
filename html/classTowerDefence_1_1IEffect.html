Detailed Description

template<typename OwnerT>
class TowerDefence::IEffect< OwnerT >

Шаблонный интерфейс для эффектов, применяемых к игровым объектам.

Определяет базовые операции для всех эффектов в игре: применение, удаление, обновление и клонирование. Эффекты могут временно изменять характеристики объектов (скорость, урон, здоровье и т.д.).

Template Parameters
OwnerTТип объекта, к которому применяется эффект.
