Detailed Description

template<typename OwnerT>
class TowerDefence::EffectComponent< OwnerT >

Шаблонный компонент для управления эффектами объекта.

Инкапсулирует коллекцию эффектов (IEffect) и предоставляет методы для их добавления и обновления. Автоматически применяет эффект при добавлении и обновляет все эффекты каждый кадр.

OwnerTТип объекта-владельца эффектов.
