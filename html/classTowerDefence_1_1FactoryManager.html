Detailed Description

template<typename IBuildResult, typename IFactoryT>
class TowerDefence::FactoryManager< IBuildResult, IFactoryT >

Менеджер фабрик для создания и восстановления игровых объектов.

Управляет коллекцией фабрик, предоставляя единый интерфейс для создания объектов различных типов с поддержкой dependency injection для фабрик.
