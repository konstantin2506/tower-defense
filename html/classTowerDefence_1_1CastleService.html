Detailed Description

Сервис управления замком.

Финальная реализация ICastleService, координирующая взаимодействие между репозиторием замка, экономикой и сервисом идентификаторов. Обеспечивает бизнес-логику для улучшения, ремонта и управления состоянием замка.
