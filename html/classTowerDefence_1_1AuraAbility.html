Detailed Description

template<typename OwnerT, typename ServiceT, typename EffectT>
requires CanAddEffect<ServiceT, OwnerT> && CanGetInRadius<ServiceT>
class TowerDefence::AuraAbility< OwnerT, ServiceT, EffectT >

Способность ауры, которая воздействует на объекты в радиусе.

Применяет эффект к объектам в определенном радиусе от владельца с заданным интервалом времени.

Template Parameters
OwnerTТип владельца ауры.
ServiceTТип сервиса для объектов, на которые воздействует аура.
EffectTТип эффекта, который применяет аура. @requires ServiceT должен удовлетворять концептам CanAddEffect и CanGetInRadius.
